Research · Deep Learning Pipeline

SDO Solar Image Pipeline & Predictive AI

Context

Standard terrestrial ML models fail on astronomical data due to radiometric constraints and missing spatial dynamics. Space weather forecasting required a custom architecture capable of processing 11 years of raw NASA satellite imagery.

Architecture

Engineered an end-to-end pipeline processing ~24,000 FITS files into 6-channel tensors. Modified a ResNet-34 backbone for self-supervised learning (SimCLR), preserving magnetic polarities, and injected these physics-encoded weights into YOLOv5x for feature localization.

Impact

Successfully compensated for solar fluid dynamics using the Snodgrass model and applied Granger Causality to statistically prove that Active Region expansions forecast sunspot formations with an optimal 3-day lead time.

Enterprise Android App · Offline-First

Lyra POS

"Engineering is the bridge between the precision of Physics and the chaos of human needs."
Context

F&B operations require absolute reliability. Existing cloud POS systems often fail during network drops, halting transactions, while on-premise solutions are too rigid for modern split-payment audits and real-time remote monitoring.

Architecture

Developed an enterprise-grade Android application using React Native and Supabase. Implemented an offline-first architecture utilizing AsyncStorage for background queue synchronization, and enforced strict relational data integrity (hybrid stock, bundle variants) via Supabase Stored Procedures (RPCs).

Impact

Achieved zero-latency checkouts on Android devices through optimistic UI rendering. The system seamlessly handles network outages, automatically reconciling split financial ledgers (Cash/QRIS) once connectivity is restored, providing precise audit trails without interrupting business flow.
